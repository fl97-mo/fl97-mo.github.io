# fl97-mo.github.io

Personal portfolio site with a CRT/terminal interface.

Built with React, TypeScript, Vite, Tailwind CSS, Radix Accordion, Lucide icons, and Zustand.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is written to `dist/`.

## Notes

- Music and sound files live in `public/`.
- Attribution details are in `ATTRIBUTIONS.md`.
- The site is deployed as a static Vite build.
